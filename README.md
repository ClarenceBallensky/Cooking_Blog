# Cooking Blog
A modern, responsive cooking blog web application that allows users to customize ingredients and dynamically scale portions in a clean, intuitive interface.

This project was built to demonstrate component-based architecture, reactive state management, and modern frontend tooling.

## ğŸ› ï¸ Installation & Setup
Dependencies:
- Node.js (v18 or higher recommended)
- npm (comes with Node)

Instructions:
1. Clone the Repository

   ```
   git clone https://github.com/ClarenceBallensky/Cooking_Blog.git
   ```

   Use the `cd` command to navigate to Cooking_Blog on your machine 
2. Install Dependencies
   ```
   npm install
   ```
   This will automatically install:
   - SvelteKit
   - Tailwind CSS
   - shadcn
   - All other required dependencies

3. Run Development Server

   ```
   npm run dev
   ```
   Open up your browser of choice and navigate to
   ```
   http://localhost:5173
   ```
